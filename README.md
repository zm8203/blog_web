# 八代博客前端项目

## 项目介绍

一些和其他博客不一样的点

1. 文章数据直接存储到es数据库，减少同步问题，使用redis缓存点赞数据，浏览量，所以文章随便点赞
2. 文章全文搜索，可精确定位到某一个自然标题处 
3. 系统配置项众多，满足定制化需求 
4. 日志系统，登录，操作，运行日志，可帮助管理员更好的观察系统运行情况 
5. 群聊功能及其群聊配置 
6. 八代具有完备的原型设计
